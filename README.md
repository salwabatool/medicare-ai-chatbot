# Medicare AI Chatbot

## Project Overview
The Medicare AI Chatbot is designed to assist users in navigating Medicare-related inquiries through an intelligent conversational interface. The chatbot aims to enhance user engagement and provide accurate information regarding Medicare services, eligibility, and enrollment.

## Features
- **User-friendly interface**: Easy to interact with through chat.
- **Natural Language Processing**: Understands and responds to user queries.
- **Information Retrieval**: Provides accurate answers based on user input.
- **Multi-Platform Support**: Can be deployed on various platforms (web, mobile).

## Model Details
The chatbot is powered by a state-of-the-art language model trained on a variety of healthcare-related datasets, specifically optimized for Medicare information.

## Dataset Information
The training dataset includes:
- Publicly available Medicare data.
- User interaction logs (anonymized).
- FAQs and guidelines sourced from official Medicare resources.

## How to Use It
1. Launch the application.
2. Initiate a conversation by typing your query in the chat interface.
3. The chatbot will respond with relevant information.
4. For complex queries, it may suggest speaking to a live representative.

## Technical Stack
- **Backend**: Python, Flask
- **Frontend**: React
- **Database**: PostgreSQL
- **Deployment**: AWS
- **Machine Learning**: Hugging Face Transformers

## Installation Instructions
1. Clone the repository: `git clone https://github.com/salwabatool/medicare-ai-chatbot.git`
2. Change directory: `cd medicare-ai-chatbot`
3. Install dependencies: `pip install -r requirements.txt`
4. Set up the database with provided scripts.
5. Run the application: `python app.py`
